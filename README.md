### 1. 🔄 Customer Churn Prediction
**Folder:** [`Customer Churn/`](./Customer%20Churn)

| Detail | Info |
|--------|------|
| **Goal** | Predict which customers are likely to cancel their service |
| **Algorithm** | Random Forest Classifier |
| **Dataset** | Telecom customer service data |
| **Key Metrics** | Accuracy, Precision, Recall, F1-Score, ROC-AUC |

**What I did:**
- Performed comprehensive EDA to identify key churn drivers (tenure, contract type, monthly charges)
- Handled missing values, encoded categorical variables, and scaled features
- Built and tuned a Random Forest model (n_estimators, max_depth hyperparameters)
- Visualized feature importance to provide clear business recommendations
- Achieved strong ROC-AUC score with optimized precision-recall balance

**Tools:** `Python` `Pandas` `NumPy` `Scikit-learn` `Matplotlib` `Seaborn`

### 2. 🎬 Netflix Movie Recommendation System
**Folder:** [`Netflix_Movie_Recomendation/`](./Netflix_Movie_Recomendation)

| Detail | Info |
|--------|------|
| **Goal** | Generate personalized movie recommendations for users |
| **Algorithm** | SVD (Singular Value Decomposition) — Collaborative Filtering |
| **Dataset** | User-movie rating matrix |
| **Key Metrics** | RMSE (Root Mean Square Error) |

**What I did:**
- Processed and cleaned user-movie rating data
- Applied SVD matrix factorization to identify latent user and item factors
- Predicted user preferences for unseen movies and ranked recommendations
- Optimized SVD parameters for minimum RMSE
- Validated recommendation quality with cross-validation

**Tools:** `Python` `Pandas` `NumPy` `Scikit-learn`

### 3. 📈 Google Stock Price Forecasting
> *(Notebook coming soon)*

| Detail | Info |
|--------|------|
| **Goal** | Forecast Google's future stock prices using historical data |
| **Algorithm** | ARIMA & SARIMA (Time Series) |
| **Dataset** | Google historical stock price data |
| **Key Metrics** | AIC, RMSE |

**What I did:**
- Performed stationarity testing using the ADF (Augmented Dickey-Fuller) test
